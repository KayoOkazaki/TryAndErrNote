RWD設定ファイルの読込▲Top

スマホ対応

レスポンシブWEBデザイン（RWD）が設定済みのCSSファイルを、
media属性に指定した条件により読込を行う。
以下例では横幅が最大480pxまでのスマホ画面に、rwd.CSSファイルを適用する指定となる

HTML記述

<head>
    <link rel="stylesheet" media="screen and (max-width: 480px)" href="css/rwd.css" />
    省略
</head>

telの自動変換機能を無効▲Top

スマホ表示時の電話番号の自動リンクを外す

スマホ表示では文書中の電話番号をタップすると自動でかける機能があるが
数字の羅列も電話番号と誤認識し、意図しない数字にリンクが設定されてしまう。
この機能を無効にして任意の電話番号だけリンクを付ける方法がある。

telの自動変換機能を無効

HTML記述

<head>
    <meta name="format-detection" content="telephone=no">
    省略
</head>

任意の電話番号だけリンクを付ける

HTML記述

<a href="tel:0312345678">03-1234-5678</a>

スマホ画面の幅の倍率を原寸に指定▲Top

スマホ仮想ウィンドウサイズを変更

viewportはスマホで表示する仮想ウインドウのサイズ（初期値980px）を指定する属性。
この指定を初期値980pxのままでPCサイトをスマホで見ると、ページ全体が縮小され
スマホ画面にピッタリのサイズに収まるように表示される。
そのままの状態では文字は細かくて読みづらく、また正しい場所をタップすることが困難な状態になる。
そのためいったん原寸大の100％表示をした上で、表示しやすい設定を行う必要がある。
この原寸大100％に設定する方法が以下である。

HTML記述

<head>
    <meta name="viewport" content="width=device-width,initial-scale=1">
    省略
</head>

設定は以下の通りhref属性に表示したいicoファイルの相対パスを指定する。

HTML記述

    <link rel="icon" type="image/ico" href="images/tmfav39705.ico">
    省略
</head>
