境界線の太さ border-width▲Top

  • 継承なし　書式 ⇒ border-width: 値;

    • ＜値＞px, thin(細い), medium(普通), thick(太い)
    ※基本ｐｘで細かい指定をするため、px以外で指定する場面はほとんどない。
    必須のプロパティBorder-styleが指定されていないと、境界線は表示されない

  • 表示例
    • ◆ div{border-width:1px;}

    ◆ div{border-width:2px;}

    ◆ div{border-width:3px;}

    ◆ div{border-width:thin;}

    ◆ div{border-width:medium;}

    ◆ div{border-width:thick;}

境界線のデザイン border-style▲Top

  • 継承なし　書式 ⇒ border-style: 値;

    • ＜値＞
    solid：実線　none：非表示(初期値)　hidden：非表示(重複時優先)　double：2本線　dotted：点線
    dashed：破線　outset：隆起した立体影　inset：くぼんだ立体影　ridge：隆起した線　groove：くぼんだ線

  • 表示例
    • ◆ div{border-style:solid;}

    ◆ div{border-style:none;}

    ◆ div{border-style:hidden;}

    ◆ div{border-style:double;}

    ◆ div{border-style:dotted;}

    ◆ div{border-style:dashed;}

    ◆ div{border-style:outset;}

    ◆ div{border-style:inset;}

    ◆ div{border-style:ridge;}

    ◆ div{border-style:groove;}

境界線の色 border-color▲Top

  • 継承なし　書式 ⇒ border-color: 値;

    • ＜値＞カラーコード（16進）　色名

  • 表示例
    • ◆ div{border-color:#ff0000;}

    ◆ div{border-color:red;}

境界線一括指定 border▲Top

  • 継承なし　書式 ⇒ border: 値 値 値 ...;

    • ＜値＞border-width　border-style　border-color
    ※値の順番は順不同で半角区切りで指定。
    border-styleは必須でそれ以外は省略可。省略時は初期値が適用。
    ＜初期値＞border-width：medium　border-style：none　border-color：親要素の色を継承

  • 表示例
    • ◆ div{border:5px dotted blue;}

    ◆ div{border:solid;}⇐ 省略

境界線各辺の指定 top/right/bottom/left▲Top

  • 継承なし　書式 ⇒ border-top: 値; / border-right: 値; / border-bottom: 値; / border-left: 値;

    • ＜値＞border-width　border-style　border-color
    ※各辺の境界線を一括指定する
    border-top（上部線）、border-right（右側線）、border-bottom（下部線）、border-left（左側線）

  • 表示例
    • ◆ div{
    border-left: 5px solid orange;
    border-bottom: 1px dotted #333;
    border-top: 3px solid #333;
    border-right: 5px solid blue;
    }

境界線の角 border-radius▲Top

  • 継承なし　書式 ⇒ border-radius: 値 値 値 値;

    • ＜値＞px
    ※各角の滑らかさを一括指定する。
    値の設定順：左上 右上 右下 左下。指定なしは0にする。

  • 表示例
    • ◆ div{border-radius: 0 0 5px 10px}

  • border-radiusで円形を描画
    • ◆ div{width: 100px; height: 100px;
    background-color: red;
    border-radius:50px;}

境界画像プロパティの個別指定 source/slice/outset/repeat▲Top

  • 境界画像のパス 書式 ⇒ border-image-source: 値;

    • ＜値＞ url("xxxx.jpg")
    ※境界線の代わりに画像を表示します。表示する画像のパスを指定。

  • 境界画像の位置の割り当て 書式 ⇒ border-image-slice : 値;

    • ＜値＞ ピクセル数値
    ※各辺の分割位置（単位なし）

  • 境界画像の外側配置 書式 ⇒ border-image-outset : 値;

    • ＜値＞ px
    境界画像の外側配置表示領域を拡大

  • 境界画像の外側配置 書式 ⇒ border-image-repeat : 値;

    • ＜値＞

    repeat:
    画像がSliceで指定されたサイズで繰り返しタイル状に囲む。
    縦横のサイズがSliceした数値で割り切れない場合は画像が崩れる
    round
    画像がSliceで指定されたサイズで繰り返しタイル状に囲む。
    縦横のサイズがSliceした数値で画像がきれいに並ぶように拡大縮小する
    stretch
    繰り返しなしで画像が引き延ばされて周りを囲む。
    space
    完全に収まる最大回数の繰り返しを行った後
    画像を等間隔に隙間を空けて並べます。
    stretch repeat space
    境界画像の外側配置表示領域を拡大

  • 表示例 1.

    • 元画像
    縦300px 横300px    		 slice:100
    repeat:repeat    		 slice:100
    repeat:round    		 slice:100
    repeat:stretch    		 slice:100
    repeat:space

境界画像一括指定 border-image▲Top

  • 継承なし　境界画像一括指定　書式 ⇒ border-image: source　slice / width / outset repeat;

    • ＜値＞

    source:
    初期値：none　url("xxx.jpg")
    　
    slice:
    初期値：100％　単位なしピクセル数値　
    width:
    初期値：1px　sliceした画像のサイズを指定する px　省略可
    outset:
    初期値：0px　px　省略可
    repeat:
    初期値：stretch　repeat, round, stretch, space
  • 表示例

    • div{
    width:80px;height:80px;
    border: solid 25px transparent;
    border-image: url("../images/image-1.jpg") 100 /20px/0px round;
    }
