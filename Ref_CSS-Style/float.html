回り込み～解除までのステップ▲Top

１．ボックス（ブロック要素）をただ並べただけの表示

ブロック要素のdiv要素をhtmlで並べると、ブラウザ表示の結果はどんどんボックスが下に積み重ねられ下方向に広がっていく。
レイアウト上、メインとサブで記事をブロックごとに左右に配置し、フッターを一番下に表示したい場合、
メインとサブを横並びに設定する必要がある。

HTML記述 

<div class="box-1"><p>メイン</p></div>
<div class="box-2"><p>サイド</p></div>
<div class="box-3"><p>フッター</p></div>

CSS記述 

.box-1 { 
    width:68%; height:60px; 
}
.box-2{
    width:30%; height:60px; 
}
.box-3{
    height:25px;
}

２．froatプロパティを使用しメインとサイドのみを横に表示させたいのだが...

フッターはそのままでfloatプロパティを使用してメインを左寄せ、サイドを右寄せに設定する。
すると、floatしたメインとサブが存在しないものとみなし、フッターが上の方にあがる。

HTML記述 

<div class="box-1"><p>メイン</p></div>
<div class="box-2"><p>サイド</p></div>
<div class="box-3"><p>フッター</p></div>

CSS記述 

.box-1{
    width:68%; height:60px; 
    float:left; }

.box-2{
    width:30%; height:60px; 
    float:right; }

.box-3{
    height:25px; }

結果を表示する

３．回り込みを解除し、フッターの上にマージンをつけたい

clearプロパティを使用して、フッターに回り込みの解除を行う。
フッターの上には余白マージンを付ける場合は、Clearプロパティを付けた要素の側ではなく
float側のボックスに余白を追加する必要がある。※マージンが効かないため

HTML記述 

<div class="box-1"><p>メイン</p></div>
<div class="box-2"><p>サイド</p></div>
<div class="box-3"><p>フッター</p></div>

CSS記述 

.box-1{
    width:68%; 
    height:60px; 
    float:left; 
    margin-bottom:10px;}

.box-2{
    width:30%; 
    height:60px; 
    float:right; 
    margin-bottom:10px;}

.box-3{
    height:25px; 
    clear:both;}

文字はみ出し解除　overflow▲Top

  • 書式 ⇒ overflow:キーワード;

    • キーワード
    visible：　ボックスからはみ出して表示されます。初期値
    scroll：　入りきらない内容はスクロールして見られるようになります。
    hidden：　はみ出た部分は表示されません。
    auto：　ブラウザに依存します（一般的にはスクロールして見られるようになります）。

Clearfixのやり方▲Top

使用例

ボックス「a」の中にボックス「c」と「b」を横並びに表示させる例。
※親要素の後ろに見えない空要素を作成し回り込みを解除することで、
下にフックとして引っ掛けて高さを引き出す

HTML記述 

<div class="a clearfix">
 <div class="b"></div>
 <div class="c"></div>
</div>

CSS記述 

.a {
    width: 100%;
    padding: 0 5px 5px;
    background-color: blue;
}

.b,.c {
    height: 50px;
    background-color: #ccc;
 }

.b {width: 68%;float: left;}
.c {width: 30%;float: right;}

.clearfix::after {
 content: "";
 display:block;
 clear:both;
}
