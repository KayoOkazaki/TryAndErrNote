要素の非表示　visibilityプロパティ▲Top

  • 書式 ⇒ visibility: キーワード;

    • キーワード：visible / hidden / collapse

    visible(初期値)
    可視状態にする
    hidden
    不可視状態にするがレイアウトに領域は残る
    collapse
    行や列が不可視状態になり領域も隠れる

要素の持つ表示形式を変更　displayプロパティ▲Top

  • 書式 ⇒ display:キーワード;

    • キーワード：　inline / block / none / list-item / inline-block / table / table-cell / flex

    inline(初期値)
    インラインボックスで表示
    block
    ブロックボックスで表示(左右目一杯で改行)
    none
    不可視状態になり領域も生成しない
    list-item
    リスト項目としてマーカーを表示
    inline-block
    ブロックボックスとして生成されるが、周囲との関係はインラインボックスと同様に振る舞う
    table / table-cell
    テーブルとセルとして表示
    flex
    ボックス同士の関係を柔軟に配置

    ※非表示の場合visibility:hiddenよりdisplay:noneが使いやすい。
    ※block / inline / inline-block / tableはmarginの相殺がされない。

block値の記述と表示例▲Top

横並びでメニューを作成する例

各メニュー内の文字数が異なる場合width heightを指定するとボタン幅が決まっているので入りきらない場合は下にはみ出してしまう。
そのため、文字ごとにボタン領域が変わるように表示するにはpaddingのみを指定する。

HTML記述 

<h2>widthとheightを指定した表示例</h2>
<ul>
   <li><a class="exp1" href="a.html">あ</a></li>
   <li><a class="exp1" href="a.html">かきく</a></li>
   <li><a class="exp1" href="a.html">さしすせそ</a></li>
</ul>
<br><br><br>
<h2>paddingのみ指定した表示例</h2>
<ul>
   <li><a class="exp2" href="a.html">あ</a></li>
   <li><a class="exp2" href="a.html">かきく</a></li>
   <li><a class="exp2" href="a.html">さしすせそ</a></li>
</ul>

CSS記述 

li {list-style-type: none;
    float: left;}

.exp1 {
    border: 1px solid;
    display: block;
    width:50px;
    height:30px;
    padding: 5px 10px;}

.exp2 {border: 1px solid;
   display: block;
   padding: 5px 10px;}

inline-block値の記述と表示例▲Top

inline-blockを使用して画像を横並びにする例

ブロック要素であるdiv要素は複数表示すると、どんどん下に積み重なって表示される。
そのためfloatを使用して横並びにするが、このfloatを使わずに横並びできるので、
回り込みの解除を行うなどの表示のくずれを直す手間が省くことができる。

HTML記述 

<h2>div要素をそのまま表示</h2>
   <div><img src="../images/pc1.jpg"></div>
   <div><img src="../images/pc2.jpg"></div>
   <div><img src="../images/pc3.jpg"></div>
<br><br>
<h2>div要素をinline-blockにして表示</h2>
   <div class="exp"><img src="../images/pc1.jpg"></div>
   <div class="exp"><img src="../images/pc2.jpg"></div>
   <div class="exp"><img src="../images/pc3.jpg"></div>

CSS記述 

div img{width:100px; height:auto;}
.exp {display: inline-block;}

table/table-cell値の記述と表示例▲Top

ナビゲーション幅を固定値で表示する例

ナビゲーションの幅が決まっている場合paddingだけで各ボタン領域を決めてしまうと、
ブラウザによって文字幅の表示が変わるため全体幅が変わってしまう。
そのため、table-layout: fixed; を併用しナビゲーションの幅を均等に固定することで、
決められたナビゲーション幅でボタンサイズが均等に配分表示されるようになる。
また幅を%を指定すればスマホのような可変サイズでも柔軟に対応できる。

HTML記述 

<h2>fixedを使用しない表示例</h2>
<ul class="exp1">
   <li><a href="a.html">研修・キャリアアップ</a></li>
   <li><a href="a.html">サービス</a></li>
   <li><a href="a.html">製品</a></li>
</ul>
<br><br><br>
<h2>fixedを使用した表示例</h2>
<p>決められたナビゲーション幅でボタンサイズが均等に配分され文字を中央に配置する</p>
<ul class="exp2">
   <li><a href="a.html">研修・キャリアアップ</a></li>
   <li><a href="a.html">サービス</a></li>
   <li><a href="a.html">製品</a></li>
</ul>

CSS記述 

li {list-style-type: none;}
li {border: 1px solid; border-left: none;}
li:first-child {border-left: 1px solid;}

/* fixedを使用しない表示例 */
.exp1 {display: table; width: 80%;}
.exp1 li {
  display: table-cell;
  padding: 5px;
  text-align: center;}
.exp1 a {display: block;}

/* fixedを使用した表示例 */
.exp2 {display: table; width: 80%;
    table-layout: fixed;
}
.exp2 li {
  padding: 5px;
  display: table-cell;
  text-align: center;}
.exp2 a {display: block;}
