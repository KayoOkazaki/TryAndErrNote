自由配置　positionプロパティ　▲Top

  • 書式 ⇒ position: 値;

    • ＜値＞
    relative：左上に詰まっていく配置
    absolute：親要素を基準にした絶対位置への配置
    fixed：absoluteをベースにスクロール時も固定位置に配置
    static：位置の指定を行わない(初期値)

配置場所　top/botto/left/rightプロパティ　▲Top

  • 書式 ⇒ top: 値;　botto: 値;　left: 値;　right: 値;

    • ＜値＞ px ※マイナス数値を指定すると逆方向へ位置がズレる
    top：上からのY座標　bottom：下からのY座標
    left：左からのX座標　right：右からのX座標

    ※起点となる場所は、positionプロパティで指定された要素の各位置最端

relative（相対位置）の使用例　▲Top

前要素h2から画像の上まで20px、前要素h2から画像の左まで10px

※前要素が無い場合は親要素(ブラウザ含む)の左上が基点

HTML記述 

<h2>ゴールデンレトリバー</h2>
<img src="dog.jpg">

CSS記述 

img {
 position:relative;
 top: 20px;
 left: 10px;
}

absolute（絶対位置）の使用例　▲Top

親要素を基準にした絶対位置への配置
親要素であるブラウザの左上から画像の上まで20px、ブラウザの左端から画像の左まで10px

HTML記述 

<h2>ゴールデンレトリバー</h2>
<img src="dog.jpg">

fixed（固定位置）の使用例　▲Top

absoluteをベースにスクロール時も固定位置に配置

ブラウザを上にスクロールしても
画像は親要素に対して絶対位置で固定される

<h2>ゴールデンレトリバー</h2>
<img src="dog.jpg">

CSS記述 

img {
    position: fixed;
    top: 200px;
    left: 30px;

一般的な使用例　▲Top

  • 書式
    • 親要素 { position: relative; }
    子要素 { position: absolute; top/bottom/left/right :n }

    relativeが指定された最も近い要素を基点にして絶対位置を指定すると、
    ブラウザサイズが変わってもサイトに合わせた位置に固定されている。

    HTML記述 

    <div>
    <h2>ゴールデンレトリバー</h2>
    <img src="../images/dog.jpg"></div>
</div>

    CSS記述 

    div {width: 300px;
    height: 200px;
    border: 1px solid #333;
    position: relative;
}
img {
    position: absolute;
    top: 70px;
    left: 30px;
}

    ブラウザ表示

    結果を表示する

    ※relativeは位置指定をしてないのでabsoluteの基点としての機能しかない

    上下の重なり順　z-indexプロパティ　▲Top

    • 書式 ⇒ z-index: ;
      • positionプロパティと併用し、上下の重なり順を指定

      ＜値＞
      数値(ーマイナス値を含めた整数 単位なし)

    positionプロパティ応用例１　リストの右側に矢印を配置　▲Top

    リストの右側に矢印を配置

    疑似要素を使って要素内の後ろに空のボックス（display:block）を作成し
    relative指定した親要素のliを起点に絶対位置absoluteを指定する。

    HTML記述 

    <ul>
<li>ホーム</li>
<li>お知らせ</li>
<li>商品一覧</li>
</ul>

    CSS記述 

    li{width: 180px;
    border: 1px solid #ccc;
    padding: 5px;
    margin-bottom: 5px;
    position: relative;
}

li::after {
    content: "";
    display: block;
    width: 16px;
    height: 16px;
    background: url("../images/arrow.jpg") 0 0 no-repeat;
    background-size: contain;
    position: absolute;
    top: 8px;
    right: 15px;
}

    ブラウザ表示

    結果を表示する
